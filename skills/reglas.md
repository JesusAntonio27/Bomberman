# 📁 Reglas del directorio `/skills`

**Propósito**: Almacenar los conocimientos específicos ("skills" o habilidades) que se le inyectan al Agente de Código para configurar su rol.

## Normas:
1. **Solo de lectura (generalmente)**: Actúan como el "cerebro" o reglas base para el agente al ayudar a programar el juego o a hacer debugging.
2. **Archivos clave**:
   - `Bomberman-gameDev-Skill.md`: Directrices y detalles sobre el comportamiento de enemigos, assets, y el plan de desarrollo.
   - `debugger-Skill.md`: Pasos específicos de diagnóstico en caso de errores en `p5.js`.
