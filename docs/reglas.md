# 📁 Reglas del directorio `/docs`

**Propósito**: Almacenar toda la documentación técnica, manuales, y criterios del proyecto.

## Normas:
1. **Formato**: Todos los archivos deben estar en Markdown (`.md`).
2. **Actualización**: Si existiera un cambio en el plan de los incrementos o en las especificaciones del juego, este es el lugar oficial para reflejarlo.
3. **Contenido principal**:
   - `MVP-Bomberman.md`: Documento principal de características del producto base.
   - `GEMINI.md`: Prompt original y requerimientos estrictos para agentes e IAS.
